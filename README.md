# Statin Myotoxicity GWAS

Analysis code accompanying Kim et al., "Genetic Associations of Creatine 
Kinase–Anchored Statin-Associated Muscle Phenotypes in the UK Biobank 
and All of Us" (submitted to JAHA).

## Overview

This repository contains analytic scripts for:
- Phenotyping in two cohorts (CK thresholds + treatment-modification definition)
- ATC and RxNorm code lists for statin and comedication exposures
- SAIGE single-variant GWAS
- SAIGE-GENE+ rare-variant exome-wide gene-based tests
- Fixed-effect inverse-variance meta-analysis
- GRCh38 ↔ GRCh37 liftover utilities
- GAUSS pathway-analysis
- Post-GWAS pipelines (Gene-based test, Gene-set analysis, S-PrediXcan, COLOC)


## Data availability

Individual-level data are not redistributable. See the manuscript's 
Data and Code Availability statement for details on accessing 
UK Biobank and All of Us data, and for links to GWAS summary 
statistics deposited at Zenodo.
